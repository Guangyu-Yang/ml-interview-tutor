An interactive AI tutor powered by Claude Code that teaches using the Socratic method.

## Features

- **Socratic Method**: Asks what you know first before teaching
- **Concise Explanations**: ~200 word explanations that respect your time
- **Active Learning**: Verifies understanding with follow-up questions
- **Adaptive Teaching**: Adjusts style based on your responses
- **Personalized Experience**: Tracks your progress across sessions

## Getting Started

### Prerequisites

- [Claude Code](https://claude.ai/code) or Claude CLI
- Basic terminal/command line knowledge

### Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/Guangyu-Yang/socratic-tutor.git
   cd socratic-tutor
   ```

2. Open with Claude Code:
   ```bash
   claude
   ```

3. Start learning! Try:
   - "Teach me about recursion"
   - "Explain how neural networks work"
   - "Help me understand Big O notation"

## How It Works

The tutor follows a structured approach:

1. **Assess**: Asks what you already know about the topic
2. **Explain**: Provides a concise explanation tailored to your level
3. **Verify**: Asks questions to check understanding
4. **Adapt**: Adjusts based on your answers
5. **Record**: Updates your learner profile for future sessions

## Customization

Edit `memory/learner-profile.md` to:
- Set your learning goals
- Specify preferred topics
- Adjust difficulty level
- Track completed topics
